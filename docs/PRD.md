# 360智企 PRD

## 1. 背景与目标

- **背景**：随着大模型技术的成熟和企业数字化转型的深入，企业对AI能力的需求从"单点尝试"转向"规模化应用"。然而，企业面临大模型接入门槛高、智能体部署复杂、团队协作困难、费用管控缺失等痛点。360智企应运而生，以大模型为技术底座、龙虾智能体为核心服务，为企业提供一站式AI数字化解决方案。
- **目标**：
  - 为企业员工提供开箱即用的AI对话与智能体服务，降低AI使用门槛
  - 为企业管理员提供配额管理、费用管控、成员权限管理能力，实现AI资源的精细化运营
  - 支持多企业、多角色协作，满足企业组织架构下的权限隔离需求
  - 提供套餐购买与计费体系，支撑商业化运营

## 3. 功能范围

### 包含

1. **官网展示页面**：产品介绍、云端龙虾介绍、应用场景、定价方案、数据统计展示
2. **用户登录系统**：360账号密码登录、短信验证码登录、扫码登录、第三方登录（QQ/微信/微博/支付宝）、用户协议确认
3. **企业工作台（控制台）**：
   - 我的首页：用量概览、套餐状态、快捷入口
   - 龙虾管理：龙虾智能体列表、创建/启停龙虾
   - AI计划：大模型调用管理、模型选择、API密钥管理
   - 用量统计：调用量趋势图、Token消耗趋势图、模型维度用量分析
4. **管理后台**：
   - 概览：活跃成员数、龙虾状态、资源包统计、调用量/Token/消费趋势、成员消费排行、产品消费排行、资源包消费排行
   - 配额管理：成员配额分配、角色配额规则、配额规则增删改查
   - 访问管理：成员访问权限审批、访问状态管理
   - 角色管理：自定义角色增删改查、角色成员数统计
5. **费用中心**：
   - 费用总览：消费趋势、余额、待付款
   - 套餐定价：套餐列表、购买套餐
   - 已购套餐：套餐状态（生效/已过期/待生效）、续费管理
   - 订单管理：订单列表、订单详情、订单状态跟踪
   - 账单管理：账单列表、账单详情
   - 收支明细：收入/支出流水
6. **组织管理**：
   - 企业信息：企业基本信息展示与编辑
   - 成员管理：成员列表、添加/移除成员、角色分配
   - 组织机构：部门树形结构管理
   - 结算单元：按部门/项目设置结算单元
7. **资源组管理**：资源组列表、资源组创建、资源管理、权限管理、密钥管理、操作审计、配额管理
8. **多企业切换**：用户可在多个企业间切换，不同企业下角色不同

### 不包含

- 大模型训练与微调功能
- 私有化部署管理
- 实时音视频通信（RTC）功能实现（仅导航入口）
- 数据大数据平台（DI）功能实现（仅导航入口）
- 国际化多语言支持（当前仅中文）
- 移动端App（当前仅PC Web）
- 暗黑模式
- 消息通知系统（站内信/邮件/推送）
- 审计日志导出
- SSO单点登录集成

## 4. 关键流程

### 主流程

1. **用户注册/登录流程**：访问官网 → 点击"免费开始" → 跳转登录页 → 选择登录方式（密码/短信/扫码） → 输入凭证 → 同意协议 → 登录成功 → 跳转企业工作台
2. **企业工作台使用流程**：登录后进入工作台 → 选择/切换企业 → 查看首页概览 → 进入龙虾/AI计划模块 → 使用AI服务
3. **套餐购买流程**：进入费用中心 → 查看定价 → 选择套餐 → 确认购买 → 生成订单 → 支付 → 套餐生效
4. **配额管理流程**：管理员进入管理后台 → 配额管理 → 创建配额规则（按角色/按成员） → 设置龙虾配额和AI配额 → 启用规则
5. **成员管理流程**：管理员进入组织管理 → 成员管理 → 添加成员 → 分配角色 → 设置配额 → 成员可开始使用

### 状态机

**用户登录状态**：
```
未登录 → 登录中 → 已登录
                  ↘ 登录失败 → 未登录
```

**套餐状态**：
```
待生效 → 生效 → 已过期
                ↘ 续费 → 待生效
```

**订单状态**：
```
待支付 → 成功
       ↘ 失败
```

**成员访问权限状态**：
```
未申请 → 审核中 → 已通过
                ↘ 已拒绝
```

**配额规则状态**：
```
待生效 → 生效 → 停用
```

## 5. 异常处理

- **接口超时**：显示Toast提示"网络开小差了"，提供重试按钮；用量统计图表显示骨架屏，超时后展示空状态+重试
- **接口报错（5xx）**：显示全屏错误页，包含错误描述和"返回首页"按钮；表单提交类操作显示Toast提示"服务异常，请稍后重试"
- **业务异常**：
  - 余额不足：弹窗提示"余额不足"，引导前往充值/购买套餐
  - 权限不足：Toast提示"您没有权限执行此操作"，成员角色隐藏管理后台入口
  - 重复提交：按钮点击后立即禁用+loading状态，防止重复提交
  - 配额超限：Toast提示"配额已用尽，请联系管理员"，龙虾/AI调用被拦截
  - 套餐已过期：显示状态标签"已过期"，引导续费
- **数据为空**：
  - 成员列表为空：显示空状态插图+文案"暂无成员，去添加"
  - 订单列表为空：显示空状态插图+文案"暂无订单"
  - 用量数据为空：图表区域显示"暂无数据"
- **网络断开**：Toast提示"网络不可用，请检查网络连接"，页面保留最后成功加载的数据

## 6. 验收标准

- **功能验收**：
  - 游客可浏览官网所有内容（产品能力、龙虾介绍、场景、定价）
  - 用户可通过3种登录方式（密码/短信/扫码）成功登录
  - 主账号/管理员可查看管理后台所有模块
  - 成员仅可查看工作台和费用中心（无管理后台入口）
  - 套餐购买流程完整：选套餐→下单→支付→生效
  - 配额规则可按角色和按成员两种方式创建和管理
  - 多企业切换功能正常，切换后数据隔离
  - API密钥可创建、查看（脱敏）、删除
- **性能验收**：
  - 首屏加载时间 ≤ 2s
  - 页面路由切换 ≤ 500ms
  - 图表渲染 ≤ 1s
- **兼容性验收**：
  - 支持Chrome 90+、Edge 90+、Firefox 90+、Safari 15+
  - 支持分辨率 1280×720 及以上
  - 响应式适配移动端H5（基础可用）
- **数据指标**：
  - 登录成功率 ≥ 99%
  - 套餐购买转化率
  - 日活跃成员数
  - 人均AI调用量

## 7. 风险与依赖

- **技术风险**：
  - 大模型API调用延迟不稳定，可能影响用户体验
  - 用量统计数据量大时，图表渲染性能可能下降
  - localStorage存储订单数据，浏览器清除数据后丢失
- **依赖项**：
  - 360账号体系（登录认证）
  - 大模型API服务（AI对话核心能力）
  - 支付系统（套餐购买支付）
  - Supabase（数据库服务，已集成SDK）
  - AWS S3（文件存储服务，已集成SDK）
- **已知限制**：
  - 当前登录为前端模拟（localStorage），未接入真实后端认证
  - 订单数据存储在localStorage，不支持跨设备同步
  - 龙虾智能体对话功能为占位实现，未接入真实大模型
  - 费用中心支付流程为模拟，未接入真实支付网关
  - 暂不支持暗黑模式
  - 暂不支持国际化
